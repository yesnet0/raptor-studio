{# Reusable server-side filesystem picker. Any input tagged with `data-fs-picker` gets a Browse button injected after it on DOMContentLoaded. Clicking the button opens a modal that navigates the server's filesystem (via /api/fs/list) and writes the selected path back into the input on "Use this directory". Set `data-fs-picker="files"` to allow file selection in addition to directories. #}
